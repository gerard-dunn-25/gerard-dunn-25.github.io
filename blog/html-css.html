What are the best practices associated with using classes vs. ids?

16 January 2025

Classes and Ids are both attributes that you can use to refer to HTML elements and format them in your style sheet.

Classes:

  • Classes are used by creating a style sheet entry using the format “.example1”
  • Once you have added your style conditions, you refer to this entry by adding class=”example1” to your elements opening tag
  • In practice, if you wanted the first heading 1 elements font to be red for example, it may look like this:

ids:

  • An id is similar to a class in that you create a style sheet entry using the format “#example2”
  • Once you have added your style conditions, you refer to this entry by adding id=”example2” to your elements opening tag
  • If you wanted the second heading 1 element font to be green for example, it may look like this:

While they are both effectively very similar ways to do very similar format styling, there are some important differences to be aware of. While multiple html elements can share the same class, you cannot have more than one element with the same id.

Classes & Ids are both used by CSS and Javascript to refer to specific elements for styling so making sure that you are paying attention to case sensitivity is also important.

Lastly, you can also use an id to make an html bookmark which can be used as a link to jump to a certain section if your web page is rather long.

When used effectively, these are useful tools to help you design and format your webpage just how you want it to look.
